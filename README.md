# ImageSearch

## Overview

類似画像検索 API & サンプル

## Install & Settings

- `git clone` or `download zip & unzip` this repository

- `$ npm install`

    - if you got error( when installing canvas ), this can be workaround:

        - `$ npm install canvas --python=python2.7`

        - `$ npm install`

- `$ node app`
